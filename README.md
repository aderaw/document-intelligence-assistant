# 📄 Document Intelligence Assistant

An AI-powered system that extracts structured information from PDF documents using Gemini LLM and Streamlit.

---

## 🚀 Features

- Upload PDF documents
- Extract text using PyMuPDF
- AI-based structured data extraction (Gemini)
- Streamlit web interface
- Clean JSON output

---

## 🧠 Tech Stack

- Python
- Streamlit
- Google Gemini API
- PyMuPDF
- dotenv

---

## 📷 Demo

Upload a PDF → Get structured JSON instantly

```bash
pip install -r requirements.txt
